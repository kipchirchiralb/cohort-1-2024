HTML Landmarks / Structural Elements

These are tags that provide a clear structure of a web document. They help developers create more semantically meaningful and accessible web pages. They include header,nav, footer, article, main, aside, section

  1. Header - Represents the header of a section or page. It can contain headings, logos, navigation menus, and other introductory content.
  2. Nav - Defines a navigation menu, typically used within a header or a footer element
  3. Main - Represents the main content of the document. It should not include headers, footers, or sidebars(aside)
  4. Article - Represents a self-contained piece of content that could be distributed and reused independently. It can include text, images, videos, etc.
  5. Section - Represents a thematic grouping of content, typically with a heading. It helps in organizing content into meaningful sections.
  6. Aside -Represents content that is tangentially related(digressive/irrelevant) to the content around it, like sidebars or pull quotes.
  7. Footer - Represents the footer of a section or page. It often contains metadata, copyright information, or links to related documents.
