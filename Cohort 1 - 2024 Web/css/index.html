CSS - Cascading Style Sheet

Where do we write our css

  • Inline CSS -- using the style attribute to style an element
  • Internal CSS -- using the style element(preferably placed within the head element)
  • External CSS -- writing styles in a separate .css file(most appropiate way)

CSS Syntax (How do we write CSS)

  • Selector
  • Declaration Block
  • Declaration-- Property+Value
