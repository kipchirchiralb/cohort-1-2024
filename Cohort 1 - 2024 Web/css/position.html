The position property in CSS

The position property is used to control the positioning of an element within its containing element. The position property can take several values, each of which determines how the element is positioned on the web page. These values include static, relative, absolute,fixed, sticky

By default every element in our page is position statically(position: static)

  • Relative: allows us to reposition an element in relation to its static/original position
  • Absolute: allows us to reposition an element in relation to the first parent/ancestor that is not statically positioned(relative,fixed,sticky -- a common usecase is making a parent relatively positioned), if no ancestor is iss either relative,fixed, or sticky, the element will be repositioned in relation to the viewport/screen
  • Fixed : allows us to reposition an element in relatin to the viewport/screen
  • sticky: The element will remain static but stick to an indicated position in relation to the viewport
