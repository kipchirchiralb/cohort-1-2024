CSS Flexbox

help

Flexbox is a layout model in CSS that allows you to design complex layouts more efficiently and with less code than traditional models. Its is useful in building responsive applications.

To use flexbox, you make an element a flex container, which then allows you to lay out its children(not descendants) in rows and/or columns

In most cases a div(division) element is used as a container
